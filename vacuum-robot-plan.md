## Track A: 嵌入式 STM32（下位机）

> **开发环境**：Windows（Keil + 源码 + 接线图）
> **学习载体**：江科大 STM32 基础课程 → 平衡车项目（已购整套硬件，调试中）

平衡车项目已覆盖扫地机下位机的大部分核心技能：PWM 电机控制、MPU6050 姿态解算、PID 闭环、编码器。调通平衡车 = STM32 基本功过关。

### Phase A1 — 完成平衡车项目 🔄 进行中
- 江科大 STM32 基础课程（GPIO/PWM/编码器/UART/I2C/ADC）
- 平衡车硬件组装 + 接线
- MPU6050 姿态解算（DMP 或 Mahony 互补滤波）
- 直立环 + 速度环 PID 调参
- 蓝牙/遥控调试

**验证**：平衡车站立不倒，能遥控行走

### Phase A2 — 从平衡车到扫地机：电机 PID 迁移
- 将平衡车的 PID 经验迁移到差速底盘
- 增量式 PI 速度控制器
- 左右轮同步走直线 2m，测偏航偏差

**验证**：速度稳态误差 < 5%，2m 直线偏航 < 5°

### Phase A3 — Pi-STM32 串口协议
- 二进制协议帧：帧头+长度+命令+数据+校验
- 50Hz 双向通信
- STM32 端解析速度指令 + 上报传感器数据
- Pi 端 Python 收发

**验证**：10 分钟持续通信不掉帧

### Phase A4 — 扫地机专用外设
- ADC 读红外对管电压（悬崖传感器原理）
- GPIO 读碰撞传感器（微动开关）
- 电池电压 ADC 监控（低电量上报）

**验证**：所有传感器数据正确上报到 Pi
